About

BetaTetris, created by Adrien Wu, is an artificial intelligence trained to play NES Tetris. It has achieved several milestones, like winning the first Classic Tetris AI Exhibition, reaching the game crash under very harsh conditions, and becoming the first agent to reach the rebirth screen on an unmodified version of the game. Many of these accomplishments required custom fine tuning, meaning BetaTetris isn't technically just one AI, but rather a collection of agents that each excel at a specific task. Here, you can BetaTetris to analyze any position with a number of parameters, allowing you to learn by example from a world class Tetris player.
